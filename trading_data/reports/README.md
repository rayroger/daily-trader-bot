# Daily Status Reports

This directory contains daily status reports generated by the trading bot.

## Overview

The daily status reports provide a comprehensive snapshot of the trading bot's performance, including:

- **Portfolio Overview**: Current balance, positions, and trading activity
- **Performance Metrics**: Returns over various time periods (daily, weekly, monthly, overall)
- **Benchmark Comparison**: Performance comparison against SPY (S&P 500) and QQQ (NASDAQ-100)
- **Position Details**: Current holdings with profit/loss calculations

## Report Format

Reports are generated as plain text files with the naming convention:
```
status_report_YYYY-MM-DD.txt
```

## Automated Generation

Reports are automatically generated daily via GitHub Actions at 5:00 PM ET (after market close and after the trading bot runs). The workflow is defined in `.github/workflows/daily-status-report.yml`.

## Manual Generation

To generate a report manually, run:

```bash
python generate_daily_status.py
```

## Viewing Reports

Reports can be viewed:
1. Directly in this directory
2. As GitHub Actions artifacts (retained for 90 days)
3. In the commit history (reports are committed back to the repository)

## Notes

- Benchmark data requires internet access and may not be available in all environments
- Reports are generated based on the portfolio state at the time of execution
- Historical reports provide a complete audit trail of portfolio performance
